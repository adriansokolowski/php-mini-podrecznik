## Co to jest PSR
PSR (z ang. PHP Standard Recommendation) są to zalecenia pisania kodu w PHP. Ułatwiają czytanie kodu, przydatne w pracy w grupie.
```Przestarzałe - od 2014-10-21 PSR-4 jest teraz zalecany jako alternatywa.```

```php
test
```
